In this lab, I demonstrated a scenario where I had forgotten both the username and password of an Ubuntu Server virtual machine and needed to regain access.

## Steps Performed

1. Restarted the Ubuntu Server virtual machine.

2. Pressed the **Esc** key repeatedly during the boot process until the **GRUB menu** appeared.

3. Accessed the **Advanced Options for Ubuntu** menu.

4. Selected the appropriate **Recovery Mode** option for the installed Ubuntu version.

5. Entered the recovery environment to perform system maintenance tasks.

6. Accessed a root shell to inspect the system and identify existing user accounts.

7. Verified the correct username associated with the server.

8. Reset the password for the identified user account.

9. Updated the password successfully and exited the recovery environment.

10. Rebooted the server and logged in using the recovered username and newly configured password.

## Key Takeaway

Forgetting login credentials does not necessarily require rebuilding a server. Ubuntu's recovery mode provides administrative recovery options that can help identify user accounts and reset passwords when authorized access to the system is available. Understanding these recovery procedures is a useful skill for system administrators, help desk technicians, and IT professionals responsible for Linux systems.
